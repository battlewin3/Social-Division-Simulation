## 一、模型总体架构

### 1.1 模型类型与设计哲学

本模型属于**基于代理人的计算社会学模拟（Agent-Based Sociological Simulation）**。与传统ABM关注微观规则产生宏观涌现不同，本模型的焦点在于**个体代理人如何基于（被结构化的）局部信息对社会整体做出推断**。模型的一个关键特征是：代理人的推断不影响后续模拟——它是"单向的"（从结构到认知）。

### 1.2 总体数据流

```
[参数网格生成] → [代理人群体初始化] → [四阶段分层过程] → [社会网络形成] → [代理人推断] → [结果汇总]
       ↑                                                                                          ↓
       └──────────────────────── [100次蒙特卡洛模拟] ──────────────────────────────────────────────┘
```

## 二、参数空间设计（01_run.R: 第54-98行）

### 2.1 参数网格

模型通过`expand.grid()`生成全因子实验设计：

| 参数 | 基准值 | 含义 |
|------|--------|------|
| `seed` | 1:100 | 蒙特卡洛模拟种子 |
| `N_agents` | 1000 | 代理人总数 |
| `netsize` | 100 | 每个代理人的网络规模（潜在连接数） |
| `friendsize` | 0.8 | 实际连接比例（小世界网络参数） |
| `income_dist` | 'normal' | 收入分布形态 |
| `race_dist` | 0.36 | 少数种族比例（模拟美国非白人人口） |
| `beta_race_income` | 0.75 | 种族对继承性收入的效应（间接效应） |
| `beta_income_nhood` | 1 | 阶层对社区质量的效应 |
| `beta_race_nhood` | 0 或 race_indirect/10 | 社区准入中的种族歧视 |
| `beta_income_school` | 0 | 阶层对学校质量的直接效应 |
| `beta_ability_school` | 0.3 | 能力对学校质量的效应 |
| `beta_race_school` | 0 或 race_indirect/10 | 学校准入中的种族歧视 |
| `beta_nhood_school` | 1 | 社区质量对学校质量的影响 |
| `beta_income_earnings` | 0 | 阶层对最终收入的直接效应 |
| `beta_ability_earnings` | 0.3 | 能力对最终收入的效应 |
| `beta_race_earnings` | 0 或 race_indirect/10 | 劳动力市场中的种族歧视 |
| `beta_nhood_earnings` | 0 | 社区质量对收入的直接效应 |
| `beta_school_earnings` | 1 | 学校质量对收入的效应 |
| `network_formation` | 'smallworld' | 网络形成算法 |
| `network_stage` | 'random' / 'nhood' / 'school' / 'earnings' | 网络形成阶段 |
| `luck` | 1 | 随机扰动标准差 |
| `rescale` | TRUE | 是否在每阶段对变量进行标准化 |

### 2.2 参数约束

**歧视参数一致性约束**（第101-112行）：
```r
disc_vars <- c('beta_race_nhood', 'beta_race_school', 'beta_race_earnings')
# 确保三个阶段的歧视参数要么全为0（无歧视世界），要么全为正（有歧视世界）
# 且其总和等于 race_indirect * 3/10
```

这样设计的经济学直觉：直接种族效应（歧视）的总和远小于间接种族效应（通过阶层继承）。论文中基准设定为：间接效应 = 0.75，直接效应总和 = 0.225（= 0.75 × 3/10）。

### 2.3 稳健性检验参数修改（第126-170行）

当`robustness = TRUE`时，生成三种替代参数设定：

1. **直接效应 > 间接效应**：将所有歧视系数设为 `race_indirect/6`，同时将间接效应减为 `race_indirect/4`
2. **种族效应 > 阶层效应**：所有种族相关beta乘以1.5，所有阶层相关beta除以1.5
3. **阶层效应 > 种族效应**：所有阶层相关beta乘以1.5，所有种族相关beta除以1.5

对每种稳健性设定，使用较少的种子数（取基准种子的前5或10个）。

## 三、代理人初始化与四阶段分层过程

### 3.1 代理人初始化（第203-215行）

```r
agentsdf <- data.frame(
    agentid = 1:N_agents,
    race_i = as.numeric(pnorm(rnorm(N_agents)) > race_dist),  # 二值种族变量
    ability_i = rnorm(N_agents)                                # 正态分布的天赋能力
) %>% data.table
```

**技术细节**：
- `race_i`使用`pnorm(rnorm(N)) > 0.36`生成：先生成N(0,1)随机数，然后以标准正态CDF的36%分位为阈值，从而精确控制少数种族比例为36%
- `ability_i`为标准正态分布：独立于种族，模拟"天赋不因种族而分布不均"
- 能力被离散化为0-99的百分位等级（`ability_i_q`）

### 3.2 Stage 1：继承性阶层位置（第218-235行）

**模型方程**：
```
income_i = β_race_income × race_i + N(0, luck²)
```

**关键设计选择**：
- 继承性收入仅由种族和随机运气决定（不含能力，能力是天生寡赋而非继承性优势）
- 若`rescale=TRUE`，则标准化为N(0,1)
- 随后离散化为0-99的百分位等级

**社会理论含义**：这代表了"阶层再生产"的第一环——种族不平等的历史积累表现为不同种族群体继承的阶层位置差异。`β_race_income = 0.75`意味着种族是继承性收入的主要预测因子。

### 3.3 Stage 2：社区分层（第238-259行）

**模型方程**：
```
nhood_raw = β_race_nhood × race_i + β_income_nhood × income_i + β_ability_nhood × ability_i + N(0, luck²)
```

**关键特征**：
- 社区质量是连续变量（反映社区内部的不平等，如住房大小、设施质量差异）
- 社区准入主要由阶层决定（β_income_nhood = 1），少数由种族歧视决定
- 社区分配非确定性：带有随机扰动（红箭头标记）
- 社区名称（nhood_proper）通过将连续分数截断为`N_groups`个分位区间获得

**理论对应**：这模拟了当代美国的居住隔离——进入好社区需要财富（阶层），同时面临住房市场和制度中的种族歧视。

### 3.4 Stage 3：学校分层（第262-285行）

**模型方程**：
```
school_raw = β_race_school × race_i + β_ability_school × ability_i + β_income_school × income_i + β_nhood_school × nhood_raw + N(0, luck²)
```

**关键特征**：
- 学校准入受社区质量强烈影响（β_nhood_school = 1）：好社区→好学校
- 能力在学校选拔中起作用（β_ability_school = 0.3）
- 种族歧视存在于招生中

**理论对应**：模拟基于房产税的美国公立学校财政体系，以及私立学校和精英大学的选拔机制。

### 3.5 Stage 4：最终收入（第288-323行）

**模型方程**：
```
earnings_f = β_race_earnings × race_i + β_ability_earnings × ability_i + β_income_earnings × income_i + β_nhood_earnings × nhood_raw + β_school_earnings × school_raw + N(0, luck²)
```

**关键特征**：
- 最终收入主要由学校质量（β_school_earnings = 1）和能力（β_ability_earnings = 0.3）决定
- 劳动力市场中也存在种族歧视
- 继承性阶层和社区质量无独立的直接效应（已通过学校中介）
- 收入被离散化为`N_groups`个分位区间（earnings_proper）和100个百分位（earnings_f_q）

**理论对应**：模拟"文凭社会"中教育凭证对收入的支配作用，以及劳动力市场歧视。

## 四、社会网络形成机制

### 4.1 小世界网络生成（Small-World Networks）

当`network_formation = 'smallworld'`时（第327-425行），使用`igraph::sample_smallworld()`生成Watts-Strogatz小世界网络：

```r
g <- sample_smallworld(
    1,                        # 网络维度（1维环形格子）
    nrow(agentsdf),          # 网络大小
    nrow(agentsdf)/(smallw_factor*10),  # 邻域大小
    0.05                      # 重连概率
)
```

**小世界因子计算**（第197-199行）：
```r
smallw_factor <- netsize / ((friendsize * netsize) / 2)
```
当`netsize=100, friendsize=0.8`时，`smallw_factor = 100/(80/2) = 2.5`，邻域大小 = 1000/(2.5×10) = 40。这意味着每个节点在环形格子上与约40个最近邻相连，然后以5%的概率随机重连。

**网络结构的理论合理性**：小世界网络具有高聚类和短路径长度的特征，与现实社会网络相似。

### 4.2 四种网络形成模式

#### 模式1：随机/整合网络（Random Network, 第330-341行）
```r
g <- sample_smallworld(1, nrow(agentsdf), nrow(agentsdf)/25, 0.05)
networks <- lapply(1:N_agents, function(i) {
    as.vector(neighbors(g, V(g)[i]))
})
```
- 在所有N个代理人间形成单一全局小世界网络
- 代表**完全整合的社会世界**：网络中随机包含各种阶层和种族的代理人
- 作为**基准对照**（最接近"真实情况"）

#### 模式2：基于社区的网络（Neighborhood-based, 第343-368行）
```r
networks <- lapply(1:max(agentsdf$nhood_proper), function(i) {
    myagents <- agentsdf$agentid[agentsdf$nhood_proper == i]
    g <- sample_smallworld(1, length(myagents), length(myagents)/smallw_factor, 0.05)
    # 在每个社区内部独立形成小世界网络
    ...
})
```
- 在每个社区**内部**独立形成网络
- 代理人**仅**与同社区的其他代理人连接
- **因果后果**：种族和阶层在决定社区准入中的作用从代理人视野中消失

#### 模式3：基于学校的网络（School-based, 第370-395行）
```r
networks <- lapply(1:max(agentsdf$school_proper), function(i) {
    myagents <- agentsdf$agentid[agentsdf$school_proper == i]
    g <- sample_smallworld(1, length(myagents), length(myagents)/smallw_factor, 0.05)
    ...
})
```
- 在学校内部形成网络
- **因果后果**：种族、阶层、能力在决定学校准入中的作用均消失；更严重的因果遮蔽

#### 模式4：基于收入群体的网络（Earnings-based, 第397-424行）
```r
networks <- lapply(1:max(agentsdf$earnings_proper), function(i) {
    myagents <- agentsdf$agentid[agentsdf$earnings_proper == i]
    g <- sample_smallworld(1, length(myagents), length(myagents)/smallw_factor, 0.05)
    ...
})
```
- 在收入相近的群体内部形成网络
- **因果后果（最极端）**：所有先前的因果过程全部消失——代理人是在"因变量"上进行抽样

### 4.3 随机网络替代方案（第427-477行）

当`network_formation = 'random'`时，不使用小世界模型，而是从相应池中简单随机抽样。例如基于社区：
```r
mynetwork <- sample(
    agentsdf$agentid[agentsdf$nhood_proper == thisnhood & agentsdf$agentid != j],
    netsize * random_factor,
    replace = FALSE
)
```
论文强调结果由同质性（homophily）驱动而非网络拓扑结构驱动，因此对网络形成方式的具体选择不敏感。

## 五、推断引擎

### 5.1 推断循环结构（第482-628行）

对每个代理人和一个额外的"全知"观察者（agent N+1，即上帝视角/真实值）执行相同的推断：

```r
inferencesdf <- lapply(1:(N_agents + 1), function(j) {
    if (j <= N_agents) {
        myfriends <- networks[[j]]   # 代理人的网络
    } else {
        myfriends <- 1:N_agents      # 上帝视角：全样本
    }
    mydf <- agentsdf[agentsdf$agentid %in% myfriends]
    ...
})
```

### 5.2 描述性推断：不平等感知（第505-529行）

```r
tmpdf <- data.frame(
    agentid = j,
    model = c('inequality', 'totalss', 'withinss', 'betweenss', 'perpos'),
    mu = c(
        var(mydf$earnings_f),                              # 总方差 = 不平等程度
        sum((mydf$earnings_f - mean(mydf$earnings_f))^2),  # 总平方和
        sum(tapply(mydf$earnings_f, mydf$race_i,           # 组内平方和
            function(x) sum((x - mean(x))^2))),
        sum((unlist(tapply(mydf$earnings_f, mydf$race_i,   # 组间平方和
            function(x) rep(mean(x), length(x)))) - mean(mydf$earnings_f))^2),
        ifelse(j <= N_agents,
            100 * ecdf(mydf$earnings_f)(agentsdf$earnings_f[agentsdf$agentid == j]),
            NA)  # 感知的社会位置
    )
)
```

**技术说明**：
- `inequality` = Var(earnings)：代理人感知的不平等水平
- `betweenss / totalss`：种族间不平等占总体不平等的比例（联合命运感知）
- `perpos`：代理人自身在其感知的收入分布中的百分位位置

### 5.3 因果推断：标准化回归模型（第534-561行）

**完整模型（normal model）**：
```r
m <- lm(data = mydf, formula = earnings_f ~ income_i + ability_i + race_i)
```
提取系数估计、标准误、t值和p值。缺少的变量（因完全共线性未被估计）被补为NA。

**种族模型（race-only model）**（第567-590行）：
```r
mrace <- lm(data = mydf, formula = earnings_f ~ race_i)
```
这给出种族的**总效应**（= 直接效应 + 通过阶层和制度的间接效应）。

### 5.4 方差分解推断（第593-620行）

```r
# 对每个模型
rss <- sum(resid(thism)^2)                          # 残差平方和
tss <- sum((thism$model$earnings_f - mean(...))^2)  # 总平方和
ess <- sum((thism$fitted.values - mean(...))^2)     # 解释平方和
# 记录: r2, adjr2, rss, ess, tss
```

**关键计算**：
- R² = ESS / TSS（或 1 - RSS/TSS）
- 运气估计 = 1 - R² = 不能被种族、阶层、能力解释的收入方差比例

### 5.5 结果汇总与输出（第630-641行）

每个模拟运行（i）的输出包含：
- `agentsdf`：所有代理人的特征数据
- `inferencesdf`：所有代理人对所有模型的所有推断
- 输出保存为`01_run.RData`

## 六、结果分析流程

### 6.1 基准比较（02_results.R）

第96-114行进行关键的数据变换——以第N+1个代理人（全知观察者）为基准：

```r
infdf <- dcast.data.table(infdf, i + model + var ~ agentid, value.var = 'mu')
tmp <- names(infdf) == unique(loopdf$N_agents + 1)
names(infdf)[tmp] <- "baseline"   # 将上帝视角重命名为baseline
infdf$mu_bias <- infdf$mu - infdf$baseline  # 计算每个代理人的偏差
```

这一设计使得每个代理人的推断可以直接与真实值比较，计算出**偏差（bias）**。

### 6.2 再分配偏好映射（03_results_redistribution.R）

第163-216行定义了从推断到再分配偏好的映射函数：

```r
# 五个因素及其预期效应方向：
# 1. ownincome (-)：越富有越反对再分配
# 2. class_coefficient (+)：阶层效应越大越支持再分配
# 3. race_coefficient (+)：种族效应越大越支持再分配
# 4. ability_coefficient (-)：能力效应越大越反对再分配
# 5. descriptive_inequality (+)：感知不平等越大越支持再分配
# 6. luck_estimate (+)：运气成分越大越支持再分配

# 通过inverse-logit转换为0-1尺度上的偏好
phat <- 1 / (1 + exp(-1 * yhat))
```

第187-190行确保种族、阶层和能力的theta值联合变动（代表一致的政治意识形态）。

## 七、模型的技术假设与局限性

### 7.1 明确的简化假设

| 假设 | 理由 | 放松该假设的可能影响 |
|------|------|---------------------|
| 代理人模型正确设定 | 隔离事实产生偏差是"下限" | 有缺陷的模型将放大偏差 |
| 完美信息 | 排除信息不对称作为混杂因素 | 信息不对称将进一步恶化推断质量 |
| 单代 | 简化，聚焦核心机制 | 多代反馈可能产生放大或均衡 |
| 推断不影响环境 | 逻辑上先论证"先验"结论 | 反馈循环可能改变长期均衡 |
| 正态收入分布 | 简化 | 右偏分布可能影响再分配偏好的量级 |

### 7.2 模型的计算约束

- N = 1000代理人：在此规模下OLS回归可靠；100次蒙特卡洛在普通计算机上可行
- 可增大N或蒙特卡洛轮数以缩小置信区间
- 当前参数组合数有限（未完全探索参数空间）

## 八、关键技术的因果推断解释

### 8.1 "条件化后处理变量"的形式化

令真实因果结构为：
```
race → income → neighborhood → school → earnings
  └──────────────────→ earnings (direct discrimination)
```

当代理人在**给定neighborhood**的条件下做回归时：
```
E[earnings | race, income, ability, neighborhood]
```
实际上**控制了**neighborhood这一"后处理变量"——它本身是race和income的结果。这导致对race总效应的估计偏误：部分通过neighborhood中介的种族效应被"控制掉"。

### 8.2 各社会世界中被遮蔽的因果路径

| 社会世界 | 可见的因果效应 | 被遮蔽的因果效应 |
|---------|---------------|-----------------|
| 随机/整合 | 全部 | 无 |
| 基于社区 | race→school, race→earnings, class→school, ability→school | class→nhood, race→nhood, nhood→school |
| 基于学校 | race→earnings, ability→earnings | 所有前置路径（→nhood, →school） |
| 基于收入 | （几乎为零） | 全部因果过程 |

## 九、技术实现总结

本模型的技术实现体现了**计算社会学模拟**的几个关键设计原则：

1. **参数透明性**：所有参数集中于参数网格，完全可复现；
2. **蒙特卡洛不确定性量化**：100次随机种子模拟生成置信区间；
3. **上帝视角基准**：通过N+1号代理人提供反事实真实值；
4. **模块化阶段设计**：四阶段生命历程可独立修改或增删；
5. **稳健性检验系统化**：通过reparameterization而非单一敏感性分析；
6. **代码简洁性**：核心模拟逻辑约600行R代码，高度可读和可修改。

**代码仓库**：https://github.com/ausmani23/inferentialspaces
